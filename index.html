Segundo passo: Fazer o cadastro no site do GitHub

Site: GitHub

Em Sign up
Enter Your email "Entre com seu e-mail"
Create a password "Crie uma senha"
Enter a username "Entre com um nome"

Terceiro passo: Configurar o usuário no git

Comandos:

git config --global user.name "seu nome"
git config --global user.email "seu email cadastrado no GitHub"

Para confirmar a configuração use os seguinte comando
git config --list

Quarto passo: Configurar a chave SSH para o Github

Acesse o site de Documentação do GitHub nele teremos as instruções para criar uma chave SSH.
Clique em Connect with SSH -> Generate new SSH Key, escolha seu sistema operacional e siga os passos informados
Após o comando verifique na sua pasta de usuário se foi criada a pasta .SSH, entre no arquivo id.rsa.pub ou id_ed25519.pub, copie todo o texto.
Agora ir até o GitHub, lado direito em Settings -> SSh and GPG Keys-> New SSH Key
Escolha um Titulo, exemplo: "Computador do trabalho" e no campo Key cole o texto do arquivo .pub que você colou.
Clique em Add SSH Key , para salvar.
